Weather or Not

A centralized automation system to control common components and hourse operations. The website is the main user interface for controls such as heat, air condition etc. The website also uses Yahoo's weather api to monitor and suggest operation actions. The functionality is presented by physical attributes on the arduino. This project was created by team TJ2me at Hack Western.

Beta Odour

Beta Odour is a company created by 12 Lakehead University Shad Valley alumnins. Beta Odour's product, Pommair is a product aimed to promote human power transportation by targeting the natural odour people generate from exercise. It does not cover up the odour like deodorants or perfumes, instead it changes the odour into a sweet apple scent by using esterification.
